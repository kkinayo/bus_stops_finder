# Анализ транспортной доступности автобусных остановок

Геоинформационное веб-приложение для анализа транспортной доступности автобусных остановок города на основе открытых геоданных OpenStreetMap.

Проект позволяет отобразить остановки общественного транспорта на интерактивной карте, выбрать произвольную точку и определить ближайшую остановку с вычислением расстояния до неё.

---

## Функциональные возможности

- автоматическое получение автобусных остановок из OpenStreetMap
- предобработка и анализ геоданных
- интерактивная веб-карта
- кластеризация остановок
- определение типа остановки
- выбор произвольной точки пользователем
- вычисление расстояния до ближайшей остановки
- визуализация линии до ближайшей остановки

---

## Архитектура проекта

Проект реализован с разделением этапов получения, обработки и визуализации геоданных.
OpenStreetMap
↓
Overpass API
↓
stops.geojson
↓
preprocess.py
↓
stops_processed.geojson
↓
Веб-интерфейс (Leaflet)

---

## Источник данных

- **OpenStreetMap** — открытая база геоданных
- **Overpass API** — инструмент выборки объектов из OSM

Используемые теги:
- `highway=bus_stop`
- `public_transport=platform`
- `public_transport=stop_position`

## Установка и запуск

### 1. Установка зависимостей
pip install requests
### 2. Получение геоданных
python download_stops.py
### 3. Предобработка данных
python preprocess.py
### 4. Запуск веб-приложения
python -m http.server 8000
### 5. Открыть в браузере
http://localhost:8000/web/index.html
